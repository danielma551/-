# 閱讀網站

一個現代化的閱讀網站，支持上傳電子書並以句子為單位進行閱讀。

## 功能特點

- 📚 支持多種電子書格式（TXT、EPUB）
- 📖 一次顯示一句話，專注閱讀體驗
- ⌨️ 鍵盤導航支持（左右箭頭鍵）
- 📊 閱讀進度顯示
- 🎨 現代化、美觀的用戶界面

## 技術棧

- **框架**: Next.js 14 (React)
- **樣式**: TailwindCSS
- **圖標**: Lucide React
- **電子書解析**: epub2
- **語言**: TypeScript

## 安裝與運行

1. 安裝依賴：
```bash
npm install
```

2. 運行開發服務器：
```bash
npm run dev
```

3. 在瀏覽器中打開 [http://localhost:3000](http://localhost:3000)

## 使用方法

1. 點擊上傳區域或拖放文件上傳電子書（支持 .txt 和 .epub 格式）
2. 系統會自動將書本內容拆分成句子
3. 使用「上一句」/「下一句」按鈕或鍵盤左右箭頭鍵進行導航
4. 頂部顯示閱讀進度條
5. 點擊「返回首頁」可以上傳新的書籍

## 構建生產版本

```bash
npm run build
npm start
```
